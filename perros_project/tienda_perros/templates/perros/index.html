🐶 Tienda de Perros API

Documentación de endpoints - Django REST API

📖 Descripción

Esta API permite gestionar perros, incluyendo búsqueda, filtrado por edad, raza, tamaño y estado de adopción.

GET /dogs/

Lista todos los perros con filtros opcionales.

Ejemplo:

http://127.0.0.1:8000/dogs/?ordering=edad 
{
  "message": "Perros encontrados",
  "total": 3,
  "data": [...]
}

GET /dogs/{id}/

Obtiene la información de un perro específico.

http://127.0.0.1:8000/dogs/5/ 
{
  "message": "Perro encontrado",
  "data": {
    "id": 5,
    "nombre": "Max",
    "raza": "Labrador"
  }
}

GET /dogs/breed/{raza}/

Filtra perros por raza.

http://127.0.0.1:8000/dogs/breed/pug/

GET /dogs/search/{nombre}/

Busca perros por nombre.

http://127.0.0.1:8000/dogs/search/wal/

GET /dogs/adoptable/

Lista perros disponibles para adopción.

{
  "message": "Perros disponibles para adopcion",
  "total": 2,
  "data": [...]
}
